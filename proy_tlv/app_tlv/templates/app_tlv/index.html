“Bienvenidos a TeLoVendo”.

Pequeña guía para mostrar un template con django.

Entra a la carpeta donde crearas tu proyecto: cd nombre_directorio,

crea tu entorno virtual : python3 -m venv .env

Activa tu entorno virtual: source .env/bin/activate

instala django: pip install django

comprueba tu version de django: django-admin --version

crea tu proyecto django : django-admin startproject proy_tlv

Entra al directorio de tu nuevo proyecto : cd nuevo_proyecto,

correr el servidor :python3 manage.py runserver

crea tu aplicacion django: python3 manage.py startapp app_tlv

Después de crear una aplicación, también necesitamos decirle a Django que debe utilizarla.
Eso se hace en el fichero settings.py
-- ábrelo en el editor y en INSTALLED_APPS y agregar una línea que contiene:
'nombre_de_la_aplicacion_creada'

en urls de proyecto:

from django.contrib import admin
from django.urls import path, include

urlpatterns = [
path('admin/', admin.site.urls),
path('', include('app_tlv.urls')),
]

Crea un nuevo fichero vacío llamado urls.py en el directorio de tu aplicacion:

from django.urls import path
from . import views

urlpatterns = [
path('', views.index, name='index'),
]

Creemos una vista (view) en el directorio de la aplicacion:

from django.shortcuts import render
def post_list(request):
return render(request, 'blog/post_list.html', {})
